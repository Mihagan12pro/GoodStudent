Запустите оба сервера: в первом терминале `cd Backend && dotnet run` для C# API на порту 7298, во втором `cd front && node server.js` для Node.js на порту 5000. Откройте http://localhost:5000, войдите как преподаватель с email `karpov.av@mospolytech.ru` и паролем `Karpov2024!` чтобы отмечать посещаемость студентов через переключатели в интерфейсе, или как заведующий кафедрой с `kopylov@mospolytech.ru` и паролем `Kopylov2025!` чтобы управлять студентами, группами и назначать предметы преподавателям. База данных PostgreSQL доступна на localhost:3306 с логином postgres и паролем 1234567890. Все API endpoints доступны через Swagger на https://localhost:7298/swagger для экспериментов со студентами, группами и кафедрами.
