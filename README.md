Запустите оба сервера: в первом терминале cd Backend && dotnet run для C# API на порту 7298, во втором cd front && node server.js для Node.js на порту 5000. Откройте http://localhost:5000 и сначала войдите как заведующий кафедрой, чтобы настроить систему - управлять студентами, создавать группы, назначать преподавателей и предметы. После настройки данных войдите как преподаватель с email и паролем любым! чтобы отмечать посещаемость студентов через переключатели в интерфейсе. Все API endpoints доступны через Swagger на https://localhost:7298/swagger для экспериментов со студентами, группами и кафедрами.
